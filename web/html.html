У страницы всегда есть элементы html, head, body

Блочные элементы - занимают все пространство по ширене, поэтому всегда переносятся на новую строку

Инлайт элементы - занимают столько места, сколько требует содержимое. Если в строке хватает места, то они размещаются на одной строке

Чаще всего используеться следующие элементы:

< div>< /div> - Блочный элемент

< span>< /span>, < span>< /span> - Изображение, инлайт элемент

< h1>< /h1>, < h2>< /h2> < h3>< /h3> < h4>< /h4> < h5>< /h5> - Заголовки, блочные элементы

< img src="ссылка на изображение"/> - Изображение, инлайн элемент

< lable for="input-id"/>< /lable> - Текст подпись для элементов input. При клике на этот тест фокусируются сами инпуты.

< input type="text"/> - Строка ввода текста, является инлайт элемент

< input type="radio"/> - Радио кнопка, является инлайн элементом

< input type="chechbox"/> - Чекбокс, является инлайн элементом

< a href="Ссылка">< /a> - Ссылка, инлайн элементом

< button> < /button> -Кнопка, инлайн элементом

< ui> < /ui> < ui> < /ui> < ui> < /ui> - Список с маркерами, нумерованный список, элементов списка. Блочные элементы.
