У страницы всегда есть элементы html, head, body

Блочные элементы - занимают все пространство по ширене, поэтому всегда переносятся на новую строку

Инлайт элементы - занимают столько места, сколько требует содержимое. Если в строке хватает места, то они размещаются на одной строке

Чаще всего используеться следующие элементы:

< div>< /div> - Блочный элемент

< span>< /span>, < span>< /span> - Изображение, инлайт элемент

< h1>< /h1>, < h2>< /h2> < h3>< /h3> < h4>< /h4> < h5>< /h5> - Заголовки, блочные элементы

< img src="ссылка на изображение"/> - Изображение, инлайн элемент

< lable for="input-id"/>< /lable> - Текст подпись для элементов input. При клике на этот тест фокусируются сами инпуты.

< input type="text"/> - Строка ввода текста, является инлайт элемент

< input type="radio"/> - Радио кнопка, является инлайн элемент

< input type="chechbox"/> - Чекбокс, является инлайн элемент

< a href="Ссылка">< / a> - Ссылка, инлайн элемент

< button> < / button> -Кнопка, инлайн элемент

< ui> < /ui> < oi> < /oi> < li> < /li> - Список с маркерами, нумерованный список, элементов списка. Блочные элементы.

Ячейки вкладываются в строки, а строки вкладываются в заголовок и тело таблицы

< form> < / form> - Форма. Служит для отправки данных на сервер. Включает в себя элементы input для ввода данных и кнопку button для отправки данных на сервер

< link rel="stylesheet" href="url"/> - Присоединение к HTML файлу других документов, обычно это файлы CSS стилей

< script src="url" >код JS для выполнения< / script> - Присоединение к HTML файлу файлов JS скриптов или выполнение JS кода

< p>< / p> - Присоединение к HTML файлу файлов JS скриптов или выполнение JS кода
